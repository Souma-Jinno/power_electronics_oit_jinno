# 基本リニアレギュレータ（Series Regulator）

## 概要

最も基本的なシリーズレギュレータ（リニアレギュレータ）回路のシミュレーションです。バイポーラトランジスタ（BJT）と基準電圧源を用いて、入力電圧を安定した出力電圧に降圧します。

## 回路構成

### 主要素子
- **電源電圧**: V1 = 10 V
- **基準電圧**: V2 = 6 V
- **トランジスタ**: Q1 (2N2222, npn型)
- **負荷抵抗**: R1 = {R} Ω（パラメータスイープ）

### 回路の特徴
- 基準電圧源がトランジスタのベースに接続
- トランジスタが可変抵抗として動作
- 負荷抵抗を変化させて出力特性を確認

## 動作原理

### 基本動作
1. 基準電圧 V_ref = 6 V がベースに印加される
2. トランジスタのベース-エミッタ間電圧 V_BE により出力電圧が決定
3. 出力電圧: V_out = V_ref - V_BE

### キルヒホッフの電圧則（KVL）
```
V_ref - V_BE - V_out = 0
V_out = V_ref - V_BE
```

### 2N2222トランジスタの特性
- V_BE(sat) ≈ 0.6 ~ 2.0 V（典型値：約0.7 V）
- 電流増幅率 h_FE ≈ 100

## シミュレーション設定

### 過渡解析
```
.tran 1m
```
- 解析時間：1 ms

### パラメータスイープ
```
.step param R list 10 50 100
```
- 負荷抵抗 R を 10Ω、50Ω、100Ω と変化させる

## 理論計算

### R = 10Ω の場合

**想定される出力電圧**（V_BE ≈ 0.7 V と仮定）:
```
V_out = V_ref - V_BE = 6.0 - 0.7 = 5.3 V
```

**出力電流**:
```
I_out = V_out / R = 5.3 / 10 = 0.53 A
```

**コレクタ電流**:
```
I_C ≈ I_out = 0.53 A
```

**ベース電流**（h_FE = 100 と仮定）:
```
I_B = I_C / h_FE = 0.53 / 100 = 5.3 mA
```

**効率**:
```
η = (V_out × I_out) / (V_in × I_C)
  = (5.3 × 0.53) / (10 × 0.53)
  = 0.53 = 53%
```

## 観察項目

### 確認すべき波形
1. **V(n002)**: 出力電圧（負荷にかかる電圧）
2. **I(R1)**: 負荷電流
3. **V(n001)**: トランジスタのベース電圧
4. **IC(Q1)**: コレクタ電流
5. **IB(Q1)**: ベース電流

### 負荷変動時の挙動
- R = 10Ω、50Ω、100Ω で出力電圧の変化を確認
- 負荷電流が減少すると出力電圧が変化する様子を観察

## 実験課題

### 基礎課題
1. 各負荷抵抗値での出力電圧を測定し、理論値と比較せよ
2. V_BE の実測値を求めよ（V_ref - V_out）
3. 負荷電流と出力電圧の関係をグラフ化せよ

### 発展課題
1. 入力電圧 V1 を変化させた場合の出力電圧の変化を調べよ
2. 基準電圧 V2 を変化させた場合の出力電圧の変化を調べよ
3. 各負荷条件での効率を計算し、負荷変動による効率の変化を評価せよ

## この回路の問題点

1. **負荷変動の影響**
   - 負荷電流が変化すると V_BE が変化し、出力電圧が変動する
   - 負荷レギュレーション（Load Regulation）が悪い

2. **効率が低い**
   - V_in - V_out の電圧降下分が熱として消費される
   - 効率 η = V_out / V_in ≈ 50%程度

3. **基準電圧の精度**
   - 理想電圧源を使用しているが、実際には安定した基準電圧が必要

## 次のステップ

この基本回路の問題点を改善するために：
- **ツェナーダイオード**を用いた基準電圧源（seriese_regurator_zener_diode）
- **オペアンプ**によるフィードバック制御（seriese_regurator_opamp）
- **MOSFET**を用いた大電流対応（seriese_regurator_opamp_mosfet）

を学習します。

## 参考：リニアレギュレータの利点と欠点

### 利点
- ノイズが小さい（連続的な電圧変化）
- 回路構成が比較的簡単
- スイッチングノイズが発生しない

### 欠点
- 効率が低い（η = V_out / V_in）
- 発熱が大きい（放熱設計が必要）
- 降圧のみ（昇圧不可）
