# Lecture 7: リニアレギュレータのLTspiceシミュレーション

## 概要

第7回講義で学習するリニアレギュレータ（シリーズレギュレータ）の各種回路をLTspiceでシミュレーションします。基本的な構成から、実用的な高精度回路まで、段階的に学習します。

## 学習目標

1. リニアレギュレータの動作原理を理解する
2. ツェナーダイオードによる基準電圧生成を理解する
3. オペアンプによるフィードバック制御を理解する
4. BJTとMOSFETの違いを理解する
5. 負荷レギュレーションと過渡応答を評価できるようになる

## シミュレーションフォルダ構成

### 1. seriese_regurator
**基本リニアレギュレータ**
- バイポーラトランジスタ（BJT）と理想電圧源を使用
- リニアレギュレータの基本動作を学習
- 負荷変動の影響を観察

**主な学習内容**:
- 基準電圧とベース-エミッタ間電圧による出力電圧決定
- V_out = V_ref - V_BE
- 負荷変動時の出力電圧変化

### 2. seriese_regurator_zener_diode
**ツェナーダイオードを用いたリニアレギュレータ**
- 実用的な基準電圧源（ツェナーダイオード）を使用
- ツェナーダイオードの特性を学習
- 負荷変動に対する安定性の向上

**主な学習内容**:
- ツェナーダイオードの逆バイアス動作
- ツェナー電圧 V_Z の安定化効果
- 電流制限抵抗の設計
- 負荷レギュレーションの評価

### 3. seriese_regurator_opamp
**オペアンプを用いたリニアレギュレータ（BJT使用）**
- オペアンプによる高精度フィードバック制御
- 抵抗分圧により任意の出力電圧を設定
- 過渡応答の観察

**主な学習内容**:
- オペアンプのフィードバック制御
- 仮想短絡（Virtual Short）の概念
- 抵抗分圧による出力電圧設定
- V_out = V_ref × (R3 + R4) / R4
- 負荷変動に対する高い安定性

### 4. seriese_regurator_opamp_mosfet
**MOSFETを用いたリニアレギュレータ**
- パワーMOSFETによる大電流対応
- ゲート駆動電流ゼロの利点
- オペアンプの両電源動作

**主な学習内容**:
- MOSFETの電圧駆動特性
- BJTとMOSFETの比較
- ゲート駆動の利点（I_G ≈ 0）
- 低オン抵抗による高効率化
- 大電流負荷への対応

## 各回路の比較

### 出力電圧の安定性

| 回路 | 負荷レギュレーション | 特徴 |
|------|-------------------|------|
| seriese_regurator | 悪い（0.2 V程度変動） | 基本構成 |
| seriese_regurator_zener_diode | やや良い（0.1 V程度変動） | 実用的な基準電圧 |
| seriese_regurator_opamp | 非常に良い（< 0.01 V） | フィードバック制御 |
| seriese_regurator_opamp_mosfet | 非常に良い（< 0.01 V） | 高精度＋大電流対応 |

### 駆動素子の比較

| 項目 | BJT (2N2222) | MOSFET (RTQ020N05) |
|------|--------------|-------------------|
| 最大電流 | ~800 mA | 20 A |
| 駆動電流 | I_B = I_C / h_FE（数 mA） | I_G ≈ 0 A |
| 電圧降下 | V_CE(sat) ≈ 0.2 ~ 0.7 V | V_DS(on) ≈ 数十 mV |
| 駆動方式 | 電流駆動 | 電圧駆動 |
| 適用 | 小~中電流 | 大電流 |

### 効率の比較

| 回路 | 効率（V_out = 5V, V_in = 10V の場合） |
|------|--------------------------------------|
| seriese_regurator | η ≈ 50% |
| seriese_regurator_zener_diode | η ≈ 40% (ツェナー損失含む) |
| seriese_regurator_opamp | η ≈ 50% |
| seriese_regurator_opamp_mosfet | η ≈ 75% (低オン抵抗) |

## 学習の進め方

### ステップ1: 基本回路の理解
1. **seriese_regurator** を実行
2. 出力電圧が V_out = V_ref - V_BE となることを確認
3. 負荷変動による出力電圧の変化を観察

### ステップ2: 実用的な基準電圧
1. **seriese_regurator_zener_diode** を実行
2. ツェナーダイオードの動作を確認
3. 基本回路との安定性の違いを比較

### ステップ3: フィードバック制御
1. **seriese_regurator_opamp** を実行
2. 仮想短絡により V_fb ≈ V_ref となることを確認
3. 抵抗分圧による出力電圧設定を理解
4. 過渡応答を観察

### ステップ4: 大電流対応
1. **seriese_regurator_opamp_mosfet** を実行
2. MOSFETのゲート電流がゼロであることを確認
3. BJTとの違いを比較
4. 低オン抵抗による高効率化を確認

## 共通の観察項目

### すべての回路で確認すべき項目
1. **出力電圧**: 設計値との一致
2. **負荷電流**: オームの法則との一致
3. **効率**: η = (V_out × I_out) / (V_in × I_in)
4. **負荷レギュレーション**: 負荷変動時の出力電圧変化率

### パラメータスイープでの確認
各回路は `.step param R list 10 50 100` により負荷抵抗を変化させます：
- R = 10Ω → 大電流（負荷大）
- R = 50Ω → 中電流
- R = 100Ω → 小電流（負荷小）

これにより、負荷変動に対する特性を評価します。

## 実験課題

### 基礎課題
1. 各回路の出力電圧を測定し、理論値と比較せよ
2. 負荷レギュレーションを計算し、4つの回路を比較せよ
3. 効率を計算し、BJTとMOSFETの違いを評価せよ

### 発展課題
1. フィードバック抵抗比を変えて、異なる出力電圧を設定せよ
   - 目標出力：5 V、10 V、12 V

2. 入力電圧を変化させて、ラインレギュレーション（Line Regulation）を評価せよ
   ```
   Line Regulation = ΔV_out / ΔV_in × 100%
   ```

3. オペアンプの電源電圧を変えて、出力範囲への影響を調べよ

4. 過渡応答時間を測定し、各回路の応答速度を比較せよ

## リニアレギュレータの応用

### 実用回路への展開
1. **可変出力電圧電源**
   - 可変抵抗器によるフィードバック抵抗の調整
   - 0 ~ V_in の範囲で調整可能

2. **電流制限機能の追加**
   - 電流センス抵抗の追加
   - 過電流保護回路の実装

3. **熱保護機能**
   - 温度センサの追加
   - サーマルシャットダウン回路

4. **複数出力電源**
   - 各出力に独立したリニアレギュレータ
   - デジタル回路とアナログ回路の分離

### 市販の3端子レギュレータ
実際の製品では、これらの機能が1つのICに集積されています：
- **78xxシリーズ**（正電圧、固定）：例 7805 (5V)、7812 (12V)
- **79xxシリーズ**（負電圧、固定）：例 7905 (-5V)、7912 (-12V)
- **LM317**（正電圧、可変）：1.25 V ~ 37 V
- **LM337**（負電圧、可変）：-1.25 V ~ -37 V

## リニアレギュレータ vs スイッチングレギュレータ

### リニアレギュレータの利点
- ノイズが小さい
- 回路が簡単
- 外付け部品が少ない
- 応答が速い

### リニアレギュレータの欠点
- 効率が低い（η = V_out / V_in）
- 発熱が大きい
- 降圧のみ（昇圧不可）

### 使い分け
- **リニアレギュレータ**：低ノイズ、小電流、入出力電圧差が小さい用途
- **スイッチングレギュレータ**：高効率、大電流、昇圧が必要な用途

## まとめ

このLecture 7では、リニアレギュレータの基本から実用的な高精度回路まで、段階的に学習しました。

**重要なポイント**:
1. 基本動作：V_out = V_ref - V_BE
2. ツェナーダイオード：実用的な基準電圧源
3. オペアンプ：フィードバック制御による高精度化
4. MOSFET：大電流対応と高効率化

これらの知識は、電源回路設計の基礎となります。
